## Setup Instructions

### 1. Clone the Repository
```bash
git clone <repo-url>
cd <project-folder>
```

### 2. Create Virtual Environment
#### Windows
```bash
python -m venv venv
venv\Scripts\activate
```
#### Linux or WSL
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install Dependencies
```bash
pip install flask flask_socketio
```

### 4. Run the App
```bash
python app.py
```
OR
```bash
python app.py
```
Go to [App Link](http://localhost:5000/)
